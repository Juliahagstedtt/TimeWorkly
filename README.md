# React + TypeScript + Vite

## TimeWorkly

Länk till Publicerad sida:
https://timeworkly.onrender.com

Välkommen till projektet WorkTimly! 
Detta är en webbaserad applikation för att registrera och hantera arbetstid.

Det här är fullstack app där jag jobbat med både backend och frontend. 

I TimeWorkly kan man:
- registrera sig
- logga in och logga ut
- Stämpla In och Ut
- registrera arbetstid manuellt
- se en veckoöversikt över arbetad tid
- redigera registrerad arbetstid

## Installation
git clone https://github.com/Juliahagstedtt/TimeWorkly.git 
cd TimeWorkly
npm install

## Starta Backend:
npm run build-server
npm run start-server

## Starta Frontend:
npm run dev
